Landing Page Hero

Particle Drift
✦

Replace floating orbs → canvas particle field

~80 tiny dots drifting slowly. 15% gold-tinted, rest pale blue-white. No clicks required — ambient breathing energy replaces the current CSS blob animations.

↑

Headline rise + gold underline draw

Hero text fades up from translateY(16px). A thin gold line draws under "Next Step" using scaleX(0→1) after the headline settles.

◎

CTA button breathes

Gold button has a very subtle box-shadow pulse (2s loop, low amplitude). Hover: slight glow intensifies. No other movement.

B

Loading · Scanning · Matching

Morphic Lines
⊞

Replace spinner → SVG grid scan

A 300×300 SVG grid of fine lines draws itself in, corner-outward. A horizontal scan line sweeps top→bottom with a gold glow. This replaces the current CSS spinner.

▮

Character-stamp status text

"ANALYZING PROFILE…" stamps in letter by letter with a scaleY(0.3→1) per character. Each character is delayed by 40ms.

▬

Precision progress bar

Thin gold bar fills from left with cubic-bezier(0.4,0,0.2,1) over 2.4s. No easing bounce — razor-precise fill. Replaces existing bar.

C

Main App · Quiz · Results

Liquid Reveal
⊡

Quiz option cards stagger in per question

Each time a new question loads, the 4 answer cards slide up staggered (0, 60, 120, 180ms delay). Selected answer scales slightly and the card gets a gold border flash.

▦

HowItWorks cards pour in on scroll

Step cards stagger in from bottom when scrolled into view. Connecting lines between steps draw left-to-right using scaleX after the first card settles.

%

Results: staggered cards + count-up

3 career cards slide up 0, 150, 300ms apart. Each card's percentage counter counts 0→target with smoothstep easing over 900ms. Match bar fills as the number rises.

Roadmap SVG Tree

B paths · C nodes
⌇

B — Connector paths draw themselves

Each SVG elbow path uses stroke-dashoffset animation (already partially built). Enhancement: a travelling gold glow dot moves along the path just before the line completes. Staggered i × 80ms per edge.

⊡

C — Node cards slide in after their path

Each node rect fades in with translateY(8px→0) timed to fire 200ms after its connecting path completes. Staggered per-section so nodes don't all pop at once.

⊙

Section headers stamp in (B-style)

When a section expands, the header text uses the character-stamp from B (fast, ~30ms/char). Collapse uses a quick fade out.

All animations respect prefers-reduced-motion — particles stop, reveals become instant fades.
